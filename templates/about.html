About This To-Do App

Welcome to TODO by Amyy — a simple and elegant To-Do list application built using Flask and SQLite.

This app helps you stay organized by letting you create, update, and delete tasks effortlessly. Whether you're tracking daily goals, managing work tasks, or just jotting down ideas — this app is designed to keep things simple and effective.

Features:

  • Add new tasks quickly with a title and description
  • View all your tasks in one place
  • Edit or delete tasks anytime
  • Clean and responsive design using Bootstrap

Tech Stack:

  • Python (Flask Framework)
  • SQLite Database
  • Bootstrap for UI styling
